# Stellar Philippines

Stellar Philippines is a community initiative for Filipino developers, founders, and innovators building on the [Stellar network](https://stellar.org). We connect builders across the archipelago with resources, events, and a space to share knowledge and ship together.

This repository is the community website for that initiative. It is not affiliated with the [Stellar Development Foundation](https://stellar.org/foundation).

---

## Contributing

We welcome contributions from builders in the Philippines and beyond. Here's how to get involved:

### 1. Set up locally

```bash
git clone https://github.com/<your-org>/Stellar-Philippines.git
cd Stellar-Philippines
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to preview changes.

### 2. Pick something to work on

- **Events** — Add or update community events in `src/data/events.ts`
- **Blogs** — Add articles or tutorials in `src/data/blogs.ts`
- **Members** — Add community member profiles in `src/data/members.ts`
- **Content & UI** — Improve copy, layout, or components under `src/components/` and `src/pages/`

Check open issues or propose a change if you have something else in mind.

### 3. Make your changes

- Keep changes focused and match the existing code style
- Test locally with `npm run dev`
- Verify the production build with `npm run build`

### 4. Open a pull request

1. Fork the repository
2. Create a branch (`git checkout -b add-cebu-meetup`)
3. Commit your changes with a clear message
4. Push to your fork and open a PR against `main`
5. Describe what you changed and why

### 5. Stay connected

- [Stellar Developer Discord](https://discord.gg/stellardev) — Ask questions and collaborate with other Stellar devs
- [Developer docs](https://developers.stellar.org/) — Official Stellar documentation
